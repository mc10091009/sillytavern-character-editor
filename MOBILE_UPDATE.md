# 📱 手機使用和更新指南

## 💡 重要提示

**本工具完全離線可用！**
- ✅ 下載後直接打開 `home.html` 就能用
- ✅ 不需要安裝任何 App
- ✅ 不需要運行服務器
- ✅ 所有數據都在你的手機上

---

## 🎯 首次使用（超簡單）

### 步驟 1：下載項目

1. **打開手機瀏覽器**（Safari / Chrome）

2. **訪問 GitHub**
   ```
   https://github.com/你的用戶名/你的倉庫名
   ```

3. **下載 ZIP**
   - 點擊綠色的 **"Code"** 按鈕
   - 選擇 **"Download ZIP"**
   - 等待下載完成

### 步驟 2：解壓文件

**iOS：**
- 打開「文件」App
- 找到下載的 ZIP 文件
- 長按 → 選擇「解壓縮」

**Android：**
- 打開「文件管理器」
- 找到下載的 ZIP 文件
- 點擊解壓

### 步驟 3：替換舊文件

1. **備份你的角色卡**（重要！）
   - 如果你創建了角色卡，先導出保存

2. **刪除舊文件夾**
   - 刪除之前的項目文件夾

3. **移動新文件夾**
   - 將解壓後的文件夾移到你想要的位置

### 步驟 4：開始使用

1. **打開文件**
   - 找到 `home.html`
   - **用瀏覽器打開**（Safari / Chrome）
   - 就這麼簡單！

2. **添加到主屏幕**（推薦）
   - iOS：點擊分享按鈕 → 添加到主屏幕
   - Android：點擊菜單 → 添加到主屏幕
   - 現在可以像 App 一樣使用了！

3. **完成！** 🎉

---

## 🔄 更新到新版本

### 最簡單的方法（推薦）

1. **備份角色卡**
   - 打開工具
   - 導出所有角色卡為 PNG
   - 保存到相冊

2. **下載新版本**
   - 重新下載 ZIP（步驟同上）
   - 解壓到新位置

3. **開始使用**
   - 打開新的 `home.html`
   - 導入你的角色卡
   - 完成！

## 🔔 自動檢查更新

如果你已經在使用本工具，它會自動提醒你更新：

### 自動提醒功能

- ✅ 每次打開網頁自動檢查（24小時一次）
- ✅ 有新版本時右上角彈出通知
- ✅ 點擊「查看更新」看更新內容
- ✅ 然後按照上面的步驟重新下載

### 手動檢查

在網頁中：
1. 打開瀏覽器開發者工具（不同瀏覽器方法不同）
2. 在控制台輸入：`manualCheckUpdate()`
3. 按回車

## 🌟 超級技巧

### 技巧 1：雲端同步

**使用 iCloud / Google Drive：**

1. 將項目文件夾放在雲端硬碟
2. 電腦上更新
3. 手機自動同步
4. 無需手動操作！

**步驟：**
- iOS：放在「iCloud Drive」文件夾
- Android：放在「Google Drive」文件夾
- 在電腦上用 Git 更新
- 手機自動獲得最新版本

### 技巧 2：GitHub Mobile

1. **安裝 GitHub App**
   - iOS / Android 都有官方 App

2. **Watch 倉庫**
   - 打開倉庫頁面
   - 點擊 "Watch"
   - 選擇 "All Activity"

3. **接收通知**
   - 有更新時收到推送通知
   - 直接在 App 中查看更新內容
   - 然後重新下載

### 技巧 3：書籤快捷方式

1. **收藏下載頁面**
   ```
   https://github.com/你的用戶名/你的倉庫名/archive/refs/heads/main.zip
   ```

2. **需要更新時**
   - 點擊書籤
   - 直接下載最新版本
   - 無需進入 GitHub 頁面

## ❓ 常見問題

### Q: 更新會丟失我的角色卡嗎？

**A:** 不會！角色卡保存在：
- 瀏覽器本地存儲（自動保存）
- 你導出的 PNG 文件

但建議更新前：
1. 導出所有角色卡為 PNG
2. 保存到相冊或雲端
3. 更新後重新導入

### Q: 多久更新一次？

**A:** 建議：
- 每月檢查一次
- 或等待自動提醒
- 重要更新會有通知

### Q: 更新失敗怎麼辦？

**A:** 簡單解決：
1. 刪除舊文件夾
2. 重新下載 ZIP
3. 重新解壓
4. 重新打開

### Q: 可以不更新嗎？

**A:** 可以！
- 舊版本仍然可用
- 但可能缺少新功能
- 建議定期更新

### Q: 更新後設置會丟失嗎？

**A:** 不會！
- 瀏覽器本地存儲會保留
- 自動保存的草稿還在
- 但建議定期導出備份

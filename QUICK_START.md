# ⚡ 3 分鐘快速開始

## 🎯 目標：離線使用，無需安裝任何東西

---

## 💻 電腦用戶

### 步驟 1：下載

1. 點擊綠色的 **"Code"** 按鈕
2. 選擇 **"Download ZIP"**
3. 等待下載完成

### 步驟 2：解壓

- **Windows**：右鍵 → 解壓縮
- **Mac**：雙擊 ZIP 文件
- **Linux**：右鍵 → 提取

### 步驟 3：打開

1. 找到解壓後的文件夾
2. **雙擊 `home.html`**
3. 完成！🎉

---

## 📱 手機用戶

### 步驟 1：下載

1. 在手機瀏覽器打開 GitHub 倉庫
2. 點擊 **"Code"** → **"Download ZIP"**
3. 等待下載完成

### 步驟 2：解壓

- **iOS**：打開「文件」App → 長按 ZIP → 解壓縮
- **Android**：打開「文件管理器」→ 點擊 ZIP → 解壓

### 步驟 3：打開

1. 找到 `home.html`
2. **用瀏覽器打開**
3. 完成！🎉

### 步驟 4：添加到主屏幕（可選）

- **iOS**：分享按鈕 → 添加到主屏幕
- **Android**：菜單 → 添加到主屏幕

現在可以像 App 一樣使用了！

## 🔄 如何更新？

### 最簡單的方法

1. **備份你的角色卡**
   - 導出所有角色卡為 PNG
   - 保存到安全的地方

2. **下載新版本**
   - 重新下載 ZIP
   - 解壓到新位置

3. **開始使用**
   - 打開新的 `home.html`
   - 導入你的角色卡

### 自動更新（可選）

- **電腦**：雙擊 `update.bat` 或 `update.sh`
- **手機**：查看 [MOBILE_UPDATE.md](MOBILE_UPDATE.md)

## 🆘 遇到問題？

### 打不開 HTML 文件？

1. 右鍵點擊 `home.html`
2. 選擇「打開方式」
3. 選擇你的瀏覽器（Chrome、Firefox、Safari 等）

### 功能不正常？

1. 確保使用現代瀏覽器（Chrome 90+、Firefox 88+、Safari 14+）
2. 清除瀏覽器緩存
3. 重新下載項目

### 還是不行？

- 查看 [README.md](README.md) 詳細說明
- 在 GitHub 提交 Issue
- 查看 [常見問題](README.md#常見問題)

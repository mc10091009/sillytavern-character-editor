# 贡献指南

感谢你考虑为 SillyTavern 角色卡编辑器做出贡献！

## 如何贡献

### 报告 Bug

如果你发现了 bug，请：

1. 检查 [Issues](https://github.com/yourusername/sillytavern-character-editor/issues) 确认问题是否已被报告
2. 如果没有，创建一个新的 Issue
3. 提供详细的复现步骤
4. 包含浏览器版本、操作系统等信息
5. 如果可能，提供截图或错误日志

### 提出新功能

1. 先在 [Discussions](https://github.com/yourusername/sillytavern-character-editor/discussions) 讨论你的想法
2. 获得反馈后，创建一个 Feature Request Issue
3. 详细描述功能的用途和实现方式

### 提交代码

1. Fork 这个仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

### 代码规范

- 使用清晰的变量和函数名
- 添加必要的注释
- 保持代码风格一致
- 测试你的更改

### Pull Request 检查清单

- [ ] 代码已经过测试
- [ ] 更新了相关文档
- [ ] 遵循了项目的代码风格
- [ ] 提交信息清晰明了
- [ ] 没有引入新的警告或错误

## 开发环境

这是一个纯前端项目，不需要特殊的开发环境：

1. 克隆仓库
2. 用浏览器打开 `index.html`
3. 开始开发！

## 许可证

通过贡献代码，你同意你的贡献将在 CC BY-NC 4.0 许可证下发布。

## 问题？

如有任何问题，请随时在 Issues 或 Discussions 中提问。
